7.14-代码规范&PS操作

标签（空格分隔）： 暑期培训

CSS书写顺序

  1. 位置属性(position, top, right, z-index, display, float等)
  2. 大小(width, height, padding, margin)
  3. 文字系列(font, line-height, letter-spacing, color- text-align等)
  4. 背景(background, border等)
  5. 其他(animation, transition等)
    正确示范如下：
.example(
    z-index: -1; 
    display: inline-block;
    font-size: 1.5em;
    color: yellow;
    background-color: blue;
)

CSS部分属性可缩写

  • padding
  • margin

  • font

    .list-box(
    border-top: 0;*border-top-style*
    font: 100%/1.6 serif;*size+line-hegiht+font-family*
    padding: 0 1em 2em;
)

  • 去掉小数点前的”0″

  • 简写命名
  • 16进制颜色代码缩写例：#eebbcc写作#ebc
  • 连字符CSS选择器命名规范
    1.长名称或词组可以使用中横线来为选择器命名。
    2.不建议使用””下划线来命名CSS选择器（IE6兼容性、JS变量命名用）
  • 不要随意使用ID
  • 为状态选择器添加前缀
  • 常用CSS命名规范

    内容：content/container
    尾：footer

注意事项::
1.一律小写;
2.尽量用英文;
3.不加中槓和下划线;
4.尽量不缩写，除非一看就明白的单词。
